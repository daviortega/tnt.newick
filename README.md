# tnt.newick


Newick and nhx input functions for [TnT](http://github.com/emepyc/tnt)

## Installation

- npm:

```
npm install tnt.newick
```

- github:

```
git clone https://github.com/emepyc/tnt.newick
cd tnt.newick
npm install
gulp build-browser
```
